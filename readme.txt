This language is called Grass
Becauase some of y'all need to touch grass.

Functions:
push "int value" - pushs a value to the stack
pop - removes a value from the stack
print - prints the top value in stack
add/sub/mul/div - does math with two highest values in stack and pushes them
*math*2 "num1" "num2" - does math based on num1 and num2 and pushes to stack
